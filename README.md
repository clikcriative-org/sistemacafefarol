# FAROL CAFÉ - Sistema Gerenciador Financeiro

Sistema completo de gestão para cafeterias. **Live:** https://pulseone.art

## Módulos
Dashboard | Vendas POS | Estoque | Compras (NF por IA) | Precificação | Relatórios | Agente WhatsApp

## Stack
Node.js + Express | Claude API | Evolution API | Docker + Traefik | React

## Deploy
cp .env.example .env && nano .env && docker compose up -d --build
